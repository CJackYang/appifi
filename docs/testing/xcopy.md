# xcopy



xcopy是包含层级结构对象的组合状态机。



用户操作包括：

1. 创建xcopy任务
2. 销毁任务
3. 更新节点或节点和全局冲突设置（两者可以独立实现但客户端使用该策略）
4. 重试



执行过程中的内部错误分为三类：

1. 单节点不可恢复错误
2. 结构性错误，可能是局部错误，也可以是全局错误
   1. 权限错误
   2. 文件系统变更
3. 单节点命名冲突，当前冲突策略无法解决或当前无冲突策略



xcopy操作分为copy，move，import和export四种类型。
